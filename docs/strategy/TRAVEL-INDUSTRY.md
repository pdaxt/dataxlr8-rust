## DMC Operations Software

### What DMCs (Destination Management Companies) Need
1. **Itinerary/Quotation Builder** — day-by-day trip planning with real-time costing
2. **Supplier Contract Management** — seasonal rates, room allotments, policies
3. **Group Manifests** — passenger lists, passport details, dietary prefs
4. **Hotel Contracting** — multi-language, meal plans, child policies, cancellation terms
5. **Transfer & Activity Scheduling** — vehicle assignment, timing, multi-leg
6. **Proposal/Voucher Generation** — branded PDFs, e-signatures
7. **Multi-Currency Pricing** — live FX, markup per currency
8. **Commission Tracking** — per-deal, per-supplier commission reconciliation
9. **WhatsApp Communication** — critical for India/Asia B2C and B2B comms

### Key Pain Points
- **Manual quotation process** — hours per custom itinerary
- **Supplier rate chaos** — seasonal rates in spreadsheets, out-of-date pricing
- **Rooming list hell** — manual room assignments, last-minute changes
- **No single system** — using 5+ tools (Excel, email, WhatsApp, accounting, booking)
- **No client visibility** — clients call/email for status updates

## Competitor Software

### Tourwriter (NZ-based)
- **Price:** $149/user/mo (Pro, 2-4 users)
- **Focus:** Boutique tour operators and DMCs crafting bespoke multi-day itineraries
- **Key Features:** Drag-drop itinerary builder, auto-pricing (margins, commissions, taxes), mobile-friendly digital proposals
- **Weakness:** Expensive, limited to itinerary-centric workflows

### Softrip (US-based)
- **Price:** Custom
- **Focus:** Multi-day group, package, and FIT operators
- **Key Features:** End-to-end (reservations, product, marketing, operations, accounting, reporting, KPIs, security)
- **Weakness:** Enterprise-heavy, not ideal for small DMCs

### TOOGO
- **Price:** Custom SaaS
- **Focus:** Full tour operating
- **Key Features:** Quotations, trip building, finances, CRM, logistics, bookings, CMS
- **Weakness:** Limited market presence

### Lemax
- **Price:** Custom
- **Focus:** Large tour operators globally
- **Key Features:** Package tour management, FIT, groups, transfers, activities
- **Weakness:** Complex, enterprise-only

### Tourplan
- **Price:** Custom
- **Focus:** DMCs globally (popular in Middle East, Europe, Africa, Americas)
- **Key Features:** Inbound operators, supplier contracting, multi-currency
- **Weakness:** Legacy system feel, expensive implementation

### Rezdy
- **Price:** $49-$399/mo

- **Key Features:** Online booking engine, channel manager, payments

### Trawex

## GDS, NDC & API Landscape

### The Big Shift: NDC Replacing GDS
- **American Airlines:** 80% of bookings now through NDC or direct channels
- **NDC (New Distribution Capability):** IATA-backed standard for airlines to sell full product catalog directly via API
- **7 major NDC versions** currently in active use (fragmentation problem)
- **AI integration among GDS providers:** reached 65% in 2025
- **GDS market growing:** 11.7% annually through 2032

### Key APIs for Travel Platforms
| API Type | Providers | What They Offer |
|----------|-----------|-----------------|
| **GDS** | Amadeus, Sabre, Travelport | Flights, hotels, cars (traditional) |
| **NDC** | Direct airline APIs | Full fare bundles, ancillaries |
| **Bedbanks** | Hotelbeds, WebBeds, GTA | 700K+ hotel properties |
| **Activities** | Viator, GetYourGuide | Tours, excursions |
| **Transfers** | Various | Airport/hotel transfers |
| **Channel Managers** | SiteMinder, Cloudbeds | Multi-OTA distribution |

### Integration Costs
- Basic OTA API integration: $5,000-$7,000
- Full GDS certification with NDC: $15,000-$25,000+
- Development time for major integrations: 2-6 months

## India-Specific Travel Tech

### Market Context
- India travel market: $23.7B (2024) → $70B by 2027
- IATO (Indian Association of Tour Operators): 1,700+ members
- Includes tour operators, hotels, airlines, transport companies

### What Indian DMCs/Agencies Need (Beyond Global)
1. **GST Compliance** — GST on services, TCS on overseas tours, export invoicing, input tax credit
2. **UPI/Razorpay Integration** — dominant payment methods
3. **WhatsApp Business API** — primary B2B and B2C communication channel
4. **Tally Integration** — most common accounting software in India
5. **Multi-Language Support** — Hindi, regional languages for domestic tourism
6. **Domestic Tourism Tools** — train bookings (IRCTC), state tourism packages
7. **Visa Processing Tracking** — status tracking, document collection
8. **Forex Management** — advance license, buying/selling FX for outbound tours

### Popular Tools Used by Indian Agencies
- **Excel/Google Sheets** — still the dominant "tool" for most small agencies
- **TravelPD** — Indian travel CRM
- **OTRAMS** — tour operator reservation and management
- **TripJack** — Indian flight booking API
- **ClearTrip API** — flight + hotel
- **Manually operated** — majority of 50K+ small agencies use no software

### Revenue Opportunity
- **50,000+ travel agencies** in India (estimated), most use no dedicated software
- Average software spend: ~$8,000-$8,500/year per agency
- If DataXLR8 captures 1% of Indian agencies at $99/user/mo with avg 3 users:
  - 500 agencies × 3 users × $99/mo × 12 = **$1.78M ARR** from India alone

## The Conversion Funnel (DMC Operations)

```
Inquiry → Lead Qualification → Quotation/Proposal → Negotiation → Booking Confirmation
    → Supplier Confirmations → Voucher Generation → Operations (transfers, hotels, activities)
    → Guest Communication → Post-Trip Review → Commission Settlement → Accounting
```

Every step is currently manual for most agencies. AI can automate 60-70% of this.

## Mobile
- **52% of OTA bookings** now happen on mobile

- Agents in the field need mobile access (site inspections, client meetings)
- Guest self-service (passport upload, dietary prefs) must be mobile-first
